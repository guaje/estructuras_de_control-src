Condicional Sí (if): Ejemplo

La función que permite calcular el valor absoluto de un número real es una función que recibe como parámetro de entrada un número real y retorna la distancia de ese valor al origen. La función valor absoluto en notación matemática se define como:

valorAbsoluto : \(\mathbb{R} \rightarrow \mathbb{R}\) $$􏰄(x) = \begin{cases} x, & \text{si $x \geq 0$;} \\ -x, & \text{en otro caso.}\end{cases}$$

## Condicional Sí (if): Ejemplo en C++ ```cpp #include <iostream> using namespace std; float valorAbsoluto(float x) { if(x >= 0) { return x; } else { return -x; } } int main() { float x = 0; float y = 0; cout << "Ingrese un numero: \t"; cin >> x; y = valorAbsoluto(x); cout << "El valor absoluto de " << x << " es: " << y << endl; return 0; } ```
## Condicional Sí (if): Ejemplo en Python ```python def valorAbsoluto(x): if x >= 0: return x else: return -x x = float(input("Ingrese un numero: \t")) y = valorAbsoluto(x) print("El valor absoluto de " + str(x) + " es: " + str(y)) ```
## Condicional Sí (if): Ejemplo en C++ ```cpp #include <iostream> using namespace std; float valorAbsoluto(float x) { return x >= 0 ? x : -x; } int main() { float x = 0; float y = 0; cout << "Ingrese un numero: \t"; cin >> x; y = valorAbsoluto(x); cout << "El valor absoluto de " << x << " es: " << y << endl; return 0; } ```
## Condicional Sí (if): Ejemplo en Python ```python def valorAbsoluto(x): return x if x >= 0 else -x x = float(input("Ingrese un numero: \t")) y = valorAbsoluto(x) print("El valor absoluto de " + str(x) + " es: " + str(y)) ```

Condicional Sí (if): Ejemplo

Una función que permite determinar el máximo de dos números reales, se puede definir como:

maximo : \(\mathbb{R} \times \mathbb{R} \rightarrow \mathbb{R}\)

aquí se tienen dos casos, si el número \(a\) es mayor que \(b\) el valor máximo es \(a\); en otro caso se debe retornar \(b\). En notación matemática esto puede ser escrito de la siguiente forma:

maximo : \(\mathbb{R} \times \mathbb{R} \rightarrow \mathbb{R}\) $$􏰄(a, b) = \begin{cases} a, & \text{si $a > b$;} \\ b, & \text{en otro caso.}\end{cases}$$

## Condicional Sí (if): Ejemplo en C++ ```cpp #include <iostream> using namespace std; float maximo(float a, float b) { if(a > b) { return a; } else { return b; } } int main() { float a = 0; float b = 0; float y = 0; cout << "Ingrese un numero: \t"; cin >> a; cout << "Ingrese un numero: \t"; cin >> b; y = maximo(a, b); cout << y << " es el mayor de los 2 numeros." << endl; return 0; } ```
## Condicional Sí (if): Ejemplo en Python ```python def maximo(a, b): if a > b: return a else: return b a = float(input("Ingrese un numero: \t")) b = float(input("Ingrese un numero: \t")) y = maximo(a, b) print(str(y) + " es el mayor de los 2 numeros.") ```
## Condicional Sí (if): Usando el operador ternario
## Condicional Sí (if): Ejemplo en C++ ```cpp #include <iostream> using namespace std; float maximo(float a, float b) { return a > b ? a : b; } int main() { float a = 0; float b = 0; float y = 0; cout << "Ingrese un numero: \t"; cin >> a; cout << "Ingrese un numero: \t"; cin >> b; y = maximo(a, b); cout << y << " es el mayor de los 2 numeros." << endl; return 0; } ```
## Condicional Sí (if): Ejemplo en Python ```python def maximo(a, b): return a if a > b else b a = float(input("Ingrese un numero: \t")) b = float(input("Ingrese un numero: \t")) y = maximo(a, b) print(str(y) + " es el mayor de los 2 numeros.") ```
